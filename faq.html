What is PubCloud?

PubCloud is a member of the CloudBrowse family, created by members of the Wilkinson Laboratory at the James Hogg iCAPTURE Centre for Cardiovascular and Pulmonary Research at the St. Paul's Hospital in Vancouver, British Columbia.

The creation of PubCloud was inspired by the recently broad adaptation of tag clouds on the Internet. PubCloud uses tag cloud as a means of summarizing large amount of abstract texts returned by the search in PubMed, a centralized repository that indexes millions of biomedical applications.


What is a tag cloud?

A tag cloud is a visually-weighted rendition of collections of words (tags) that can be used to represent the concepts present in large collections of information. The qualities of associations between each tag and the entity it describes, such as frequency or recency, are visually represented with variable font sizes and colours (e.g. more frequently used tags might be larger or brighter than less frequently used tags).

In a typical implementation of a tag cloud (i.e. Flickr), font sizes are used to represent the frequency of occurrences of the particular tag, the larger the tag, the more frequent it appears. In addition, some implementations (i.e. Connotea) use colours to represent how recent the tag was created, the brighter (i.e. reddish) the more recent. Some other implementations also add proximity, underline...etc for more measures.


How is PubCloud different from PubMed?

PubMed is a repository for biomedical literatures, in the original text format, with the addition of some structural information, such as authors, publication date, keywords, MeSH terms...etc. PubCloud, on the other hand, is a tool for summarizing information returned by the the PubMed search engine. It summarizes 3 types of data, abstract text, authors, and MeSH terms, in a tag cloud format, with frequency and recency information represented in font sizes and font colours, respectively.


Is PubCloud designed to replace PubMed?

Absolutely not! PubCloud is a way of summarizing and visualizing PubMed search results. PubCloud provides a way for quickly browsing and navigating through the tag cloud interface. By clicking on each tag you add to the list of search terms and PubCloud generates a new tag cloud using this list of terms to search PubMed. Until you are satisfied, you can then click into view the PubMed abstracts that were used to build the tag cloud.

Will PubCloud be continuously improved?

Yes, PubCloud is continuously undergoing improvements. New features will be updated as soon as they are available.


Is PubCloud a web service?

No, not yet. It will be made a web service in the near future.
